Beat Saber KBS (Knowledge Based System) to get maps recommendations based on your rank to maximize the PP (Performance Points) you can realistically gain:
insert your global Rank and the system answer with 4 maps (using the unique identifier *hashMap*) to maximize the PPs you can get.
The answer is organized as a training list with a selected intensity curve, starting with low intensity map as a warm-up and finishing with a really challenging one.

**Instructions:**
- install python
- Open terminal from the repository folder and run "pip install -r requirements.txt"
- run with python PP Maximizer Trainer (CSP).py
- instert you rank, get the best maps for your rank.

---
If you want to redownload the dataset (will take less then 2h) or train the model again run the files from 1 to 4.
Note: you **need** to install **SWI-Prolog** (64-bit) and add its folder `bin` to enviroment variables `PATH` to make it work.
